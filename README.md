# HTML-страница компании ООО "24H Софт"

Страница создана в рамках производственной практики в Университете "Синергия".

## Содержание
- Информация об организации ООО "24H Софт"
- Финансовые показатели компании
- 6 вариантов шрифтов для корпоративного стиля

## Технологии
- HTML5
- CSS3
- Google Fonts (Inter)

## Запуск
Откройте файл `index.html` в браузере.
